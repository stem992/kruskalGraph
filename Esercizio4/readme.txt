1. Recarsi all'interno di Esercizio4 e copiare il file .csv al suo interno.
2. Aprire da terminale e digitare "ant compile".
3. Successivamente digitare "ant all" per produrre gli eseguibili jar all'interno della cartella build.

Per eseguire i test:
- java -jar build/GraphDirectTest.jar

Per eseguire KruskalMain: 
- java -jar build/MainKruskal.jar "./italian_dist_graph.csv"
